# Source Images Folder

This folder contains the logo, icon, and images used throughout the website.

## Required Files

### Logo and Icon
- `logo.png` or `logo.svg` - Main website logo
- `icon.png` or `icon.ico` - Website favicon/icon (recommended: 32x32 or 64x64 for favicon, 192x192 or 512x512 for app icons)

### Home Page Images
- `hero-background.jpg` - Hero section background image (recommended: 1920x1080 or larger)
- `feature-bike-location.jpg` - Feature image for "Find Bikes Nearby" section (recommended: 800x600 or larger)
- `feature-instant-booking.jpg` - Feature image for "Instant Booking" section (recommended: 800x600 or larger)
- `feature-earn-money.jpg` - Feature image for "Earn Money as an Owner" section (recommended: 800x600 or larger)

## Usage

Images in this folder are referenced from the website using the path `/src/filename.ext`.

For example:
- Logo: `/src/logo.png`
- Hero background: `/src/hero-background.jpg`
- Feature images: `/src/feature-bike-location.jpg`
